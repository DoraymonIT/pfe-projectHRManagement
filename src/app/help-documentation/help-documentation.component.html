La premiere page que vous avez trouve c est la page d authentification dans laquelle vous pouvez saisir votre Email et mot de passe donnee par la faculte pour acceder au application et pour faire votre travail

Et voila Dirigez a nouveau dans la page d authentification et Saisiez les nouveaux donnees et vous pouvez capable d acceder a votre espace de travail .

Informations sur Les pages de l application

Cette application est une application web faire gerer les ressources humains de la FSTG en particulier les enseignat(e)s , chercheur(e)s et professeur(e)s ; Ce gestion contient plusieurs elements :

  • Fonctionnalite 1 : Gestion des employes et leurs donnees
  • Fonctionnalite 2 : Gestion de conges de ces employes
  • Fonctionnalite 3 : Gestion de permanences amninidtratives
  • Fonctionnalite 4 : Gestion de Affectation des formations,prix et punitions aux employes
  • Fonctionnalite 5 : Gestion de Affectation de Notes et L evaluation des employes
  • Fonctionnalite 6 : L etude d avancement de grade des employes
  • Fonctionnalite 7 : Gestion de L architecture Interne de la FSTG (Departements , Fonctions , Grades ,...)
  • Fonctionnalite 8 : Gestion des demandes de documents ( Attestation de travail , de Salaire )
  • Fonctionnalite 9 : Configuration et parametres ( Changer le mot de passe )
  • Fonctionnalite 10 : Historiquer de l application ( Quotidien , depuis le debut de l application )
G de l application

    • Button 3 : Supprimer : Il nous le droit de supprimer cet employe de la liste n est pas definitevement.

Architecture de l application

L application est simple , contient des interfaces simples et effective pour faire les taches de service de ressources humaines , juste cliquez sur le button ( Hamburger ) et vous pouvez capable de voir les differents fonctionnalites qui vous pouvez faire a l aide de cette application.

C est la meilleure fonctionnalite que nous avons fait durant cette application , Si vous avez du table ou liste et vous pouvez la telecharger et exporter , vous pouvez faire ca en cliquant sur le button soit EXPORTET PDF ou EXPORTER EXCEL selon votre besoin.
